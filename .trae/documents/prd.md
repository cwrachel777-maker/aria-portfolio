# 个人作品集网站 - 产品需求文档

## 1. 项目概述

**项目名称**: Aria Portfolio  
**项目类型**: 个人作品集展示网站  
**核心功能**: 展示空间设计与服装陈列复合型人才的职业经历与项目作品  
**目标用户**: 面试官、招聘方  
**设计风格**: 极简主义（黑白配色、网格布局、现代字体排版）

---

## 2. 整体结构

### 2.1 页面架构
- **导航方式**: 顶部固定导航栏 + 滚动浏览 + 点击跳转的混合导航
- **板块数量**: 5个主板块
- **响应式**: 先完成桌面端，移动端后续适配

### 2.2 导航栏设计
- **位置**: 固定于页面顶部
- **入口**: 首页 | 能力地图 | 项目经历 | 市场洞察 | 关于我
- **交互方式**:
  - 点击导航栏文字 → 直接跳转对应板块
  - 滚动浏览 → 各板块按顺序自然呈现
- **高亮状态**: 进入对应板块区域时，该导航项高亮

### Part 3: 项目经历

**内部层级结构**: 板块封面页 → 项目列表页 → 项目详情页

**返回逻辑**:
- 项目详情页点击返回 → 回到当前板块项目列表页
- 项目列表页向上滚动 → 回到板块封面页
- 点击导航栏"项目经历" → 回到 Part 3 总入口
- 点击导航栏其他入口 → 跳转到对应板块

**项目详情页顶部导航条**:
- 左侧: 返回箭头
- 中间: 当前项目名称
- 右侧: 分页指示器（多页时显示"1/2"并支持左右箭头切换）
